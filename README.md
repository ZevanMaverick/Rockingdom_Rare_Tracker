# 洛克王国异色收集器（轻量工具）

一款轻量化的《洛克王国》异色捕捉追踪工具，用于在抓捕过程中进行进度记录、成本结算与一键分享海报。

## 功能概览

- 捕获追踪（Tab1）：开始抓捕、记录事件、查看进度与球耗成本
- 精灵图鉴（Tab2）：浏览精灵列表
- 结算工具（Tab3）：球消耗成本/结算相关能力
- 分享海报：生成可保存的图片用于社群分享

## 重要说明（数据与存档）

- 本项目默认将数据保存在浏览器 `localStorage` 中：
  - 同一浏览器/同一域名下会保留
  - 更换设备或更换浏览器不会自动同步

## 本地运行

环境要求：Node.js 18+（建议）

安装依赖：

```bash
npm install
```

启动开发模式：

```bash
npm run dev
```

浏览器打开终端输出的地址（通常是 `http://localhost:5173/`）。

生产构建：

```bash
npm run build
```

本地预览构建产物：

```bash
npm run preview
```

## 规则口径（简述）

以下为产品内记录口径的简述，便于理解按钮与进度含义：

- 进度上限：80
- 混池进度仅在“随机噩梦”“星尘虫/果冻”时 +1
- “噩梦形态+1”仅影响单宠相关计数（不推动混池）
- 异色清零：
  - 目标异色：清系别/单宠（混池不清）
  - 同系异色：清系别（混池不清）
  - 随机精灵异色：清混池

> 注：实际规则以产品页面展示与后续需求更正为准。
